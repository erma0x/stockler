## How to install
1. python3 -m venv venv
1. source venv/bin/activate
1. python3 -m pip install --upgrade pip
1. pip install yfinance --upgrade --no-cache-dir


## Requirements

```
    Python >= 2.7, 3.4+
    Pandas (tested to work with >=0.23.1)
    Numpy >= 1.11.1
    requests >= 2.14.2
    lxml >= 4.5.1
    appdirs >=1.4.4
```
